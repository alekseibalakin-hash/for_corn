# Деплой: от кода до «она играет»

Хостим только статику (бэкенда нет) → держать ничего не надо, работает 24/7.
Бот уже создан. Ниже — по шагам, ~15 минут.

## Шаг 1 — запушить в свой GitHub-репозиторий
Создай на GitHub пустой приватный репозиторий (без README/лицензии). Затем в папке проекта:

```
git init
git add .
git commit -m "2048 с любовью — MVP"
git branch -M main
git remote add origin https://github.com/<USER>/<REPO>.git
git push -u origin main
```

## Шаг 2 — задеплоить на Vercel (бесплатно, HTTPS)
1. vercel.com → войди через GitHub → **Add New → Project**.
2. Импортируй свой репозиторий.
3. Vercel сам определит Vite (Framework: Vite, Build: `npm run build`, Output: `dist`) — менять ничего не надо.
4. **Deploy**. Через ~минуту будет URL вида `https://<repo>.vercel.app` — скопируй.
5. Открой URL в браузере: должна загрузиться игра (онбординг «Кукурузка»).

Дальше каждый `git push` передеплоивает автоматически. (Cloudflare Pages — равнозначная альтернатива, если предпочитаешь.)

## Шаг 3 — привязать Mini App к боту (@BotFather)
В Telegram открой **@BotFather**:
1. `/mybots` → выбери бота → **Bot Settings** → **Menu Button** → **Edit menu button URL**.
2. Вставь Vercel-URL из шага 2.
3. Текст кнопки, напр. `🌽 Играть`.

Теперь в чате бота снизу — кнопка, открывающая игру.

Опционально:
- Прямая ссылка-иконка: `/newapp` → бот → название/описание/картинка + тот же URL → получишь `t.me/<bot>/<app>`, её можно прямо отправить.
- Чтобы бот выглядел как подарок: `/setname`, `/setdescription`, `/setabouttext`, `/setuserpic`.

## Шаг 4 — проверь сам на реальном телефоне (важно)
1. Открой бота, **Start**, затем `🌽 Играть`.
2. Сделай несколько ходов → прилетит награда, её надо **«Забрать»**.
3. Используй купон → экран «покажи мужу».
4. Закрой Mini App полностью и открой снова → партия/кошелёк/прогресс на месте.
   (Сохранение — Telegram CloudStorage, работает именно внутри Telegram, не в обычном браузере.)

## Шаг 5 — отдай ей
Отправь `t.me/<botusername>` (или прямую ссылку из `/newapp`).
Скажи: «Открой, нажми Старт, потом кнопку 🌽 Играть». Всё.

## Если что-то не так
- Белый экран: открой Vercel-URL в десктоп-браузере, F12 → Console. Чаще всего — не тот URL в BotFather или деплой ещё не завершился.
- В обычном браузере прогресс может не сохраняться — это норма (CloudStorage только в Telegram; в браузере localStorage-фолбэк, в инкогнито чистится).
- Обновление игры потом: правишь → `git push` → Vercel сам передеплоит, ей переустанавливать ничего не нужно, просто откроет заново.
- Добавить награды/задания: правишь `content/*.json` → push. Движку правок не нужно.
