## Что проверит CTO-сессия после правок
- «Кукурузка» везде; нет награды до первого хода (тест + живой прогон).
- Каждая награда требует «Забрать»; очередь нескольких; тиры по сочности.
- Новый цикл: redeemed → done навсегда; expired → снова доступно; pending → без дублей.
- Победный баннер при всех completed; счётчик X/N.
- Онбординг один раз + флаг персистится; резюме из CloudStorage (не с нуля).
- Тесты/типчек/сборка зелёные; конфиг по смыслу не тронут.
